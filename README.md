# Integration Management Platform

A production-style, end-to-end full stack application built with:

- **Backend:** NestJS, TypeScript, PostgreSQL
- **Frontend:** React / Next.js (TypeScript)
- **Cloud:** AWS ECS, AWS Lambda
- **CI/CD:** GitHub Actions
- **Integrations:** Third-party APIs and webhooks

## Architecture Overview

- RESTful backend APIs running on ECS
- Event-driven background processing using AWS Lambda
- Frontend dashboard consuming typed APIs
- CI/CD pipelines for automated testing and deployment

frontend/ # Next.js frontend
